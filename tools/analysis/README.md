# Analysis Tools

This folder contains small local scripts used to inspect implementation gaps and update ranking notes across:

- `C:\dev\OpenGSCore`
- `C:\dev\OpenGSServer`
- `C:\dev\OpenGSR`

Current utilities:

- `code_analyzer.py`
- `code_analyzer2.py`
- `code_analyzer3.py`
- `debug_regex.py`
- `update_ranking.py`

These are intentionally kept as lightweight one-off helpers. If they continue to be useful, we can later split them into clearer subfolders or turn the repeated parts into shared utilities.
